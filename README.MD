# The Silent Escape

The Silent Escape adalah game survival adventure di mana pemain harus mengumpulkan item sambil menghindari zombie dalam area open world sederhana.

## Cara Install
1. Clone repository:
   git clone https://github.com/username/silent-escape
2. Buka folder project di Unity (minimal Unity 2021.3)
3. Tunggu proses import selesai
4. Klik Play untuk menjalankan game

## Fitur Saat Ini
- Movement karakter
- AI zombie sederhana
- Sistem pengumpulan item
- Map open world sederhana
